## Описание проекта и функциональность

Вы можете создавать, читать, обновлять и удалять задачи. Каждая задача содержит:

* Название
* Описание
* Приоритет (низкий / средний / высокий)
* Статус (ожидает / в работе / выполнено)
* Срок выполнения
* Время создания

Также доступны:

* Фильтрация задач по **статусу**, **приоритету** и **сроку выполнения**
* Поиск задач по **названию** или **описанию** через текстовое поле

---

## Настройка окружения

Файл `.env` находится в репозитории, так как не содержит чувствительных данных.

Существует два способа запуска проекта:

### Вариант 1: Запуск в режиме разработки

```bash
npm run dev
docker compose up postgres
```

### Вариант 2: Запуск через Docker

```bash
docker compose up --build
```

---

## Миграции базы данных

В обоих случаях необходимо выполнить миграции:

```bash
npm run migrate:dev
```

или

```bash
npm run migrate:deploy
```
